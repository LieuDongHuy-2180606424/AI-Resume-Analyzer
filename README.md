# AI Resume Analyzer & Professional Career Suite

A modern, AI-powered web application designed to help job seekers optimize their resumes, match them with job descriptions, and prepare for interviews using state-of-the-art LLMs.

---

## Key Features (Tính năng nổi bật)

### 1. **Deep AI Semantic Analysis** (Phân tích chuyên sâu)
- **Overall Match Score**: A weighted 70/30 (AI/Keywords) score indicating career compatibility.
- **Detailed Insights**: Automatically identifies strengths, critical gaps, and provides actionable improvement advice.
- **Skill Categorization**: Automatically groups your skills into Technical, Soft Skills, and Tools.

### 2. **Professional Career Suite** (Bộ công cụ sự nghiệp)
- **Resume Content Optimizer**: AI-powered rewriting of weak bullet points with professional, keyword-rich alternatives.
- **Targeted Interview Prep**: Generates specialized interview questions based on the candidate's skill gaps.
- **Interactive AI Interview Assistant**: A real-time chat interface to ask any questions about the role or interview prep.

---

## Tech Stack (Công nghệ sử dụng)

- **Frontend**: React.js (Vite), Tailwind CSS v4, Framer Motion, Lucide Icons.
- **Backend**: FastAPI (Python), PyMuPDF (Text Extraction), Spacy (NLP).
- **AI Engine**: Groq Cloud SDK (Model: **LLaMA 3.3 70B**) for near-instant responses.
